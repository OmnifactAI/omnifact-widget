JSON Configuration Example

This example shows how to configure the widget using an embedded JSON script tag. This is useful when you have many configuration options or want to keep your HTML cleaner.

Note: HTML attributes take precedence over JSON configuration. You can use both together to set defaults in JSON and override specific values with attributes.

Code

<omnifact-chat-widget>
  <script type="application/json">
    {
      "endpoint-url": "https://connect.omnifact.ai",
      "endpoint-id": "YOUR_ENDPOINT_ID",
      "title": "Help Center",
      "welcome-message": "Welcome! I'm here to help.",
      "primary-color": "#10b981",
      "position": "bottom-right"
    }
  </script>
</omnifact-chat-widget>

Mixed Configuration

You can combine JSON and attributes. Attributes will override JSON values:

<!-- primary-color will be #ef4444 (from attribute) -->
<omnifact-chat-widget primary-color="#ef4444">
  <script type="application/json">
    {
      "endpoint-url": "https://connect.omnifact.ai",
      "endpoint-id": "YOUR_ENDPOINT_ID",
      "primary-color": "#10b981"
    }
  </script>
</omnifact-chat-widget>
